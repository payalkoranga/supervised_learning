#Supervised Learning (Regression & Classification)

## Files
- `House_Price_Prediction.ipynb` – Linear Regression model for house price prediction
- `Logistic_Regression_Titanic.ipynb` – Logistic Regression model  for Titanic survival prediction
- train.csv - House Prices dataset
- titanic_train.csv -Titanic dataset

## What I Did
- Loaded a House Prices dataset and trained a Linear Regression model
- Evaluated the model using R² score
- Plotted predicted vs actual house prices
- Loaded the Titanic dataset and trained a Logistic Regression model
- Evaluated the model using accuracy score and confusion matrix

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn
